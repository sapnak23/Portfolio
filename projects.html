Azure Backup Log Automation

This project automates the daily logging of Azure Recovery Services backup jobs for multiple clients. It uses Python, Azure SDKs, Key Vault authentication, and Excel automation with `openpyxl`. It also supports Disaster Recovery status reporting.

  • Reads client and vault info from a JSON config
  • Fetches secrets securely from Azure Key Vault
  • Retrieves backup and replication status using Azure SDKs
  • Writes results to client-specific Excel sheets
